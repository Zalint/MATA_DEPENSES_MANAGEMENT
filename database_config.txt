Configuration de la base de données pour l'application:

Utilisateur: zalint
Base de données: expense_management
Hôte: localhost
Port: 5432
Mot de passe: [À définir]

Pour configurer l'application:
1. Créez un fichier .env dans le dossier racine avec:
   DB_USER=zalint
   DB_HOST=localhost
   DB_NAME=expense_management
   DB_PASSWORD=votre_mot_de_passe_zalint
   DB_PORT=5432

2. Ou modifiez directement server.js ligne 13-17 avec le bon mot de passe

3. Exécutez les scripts SQL:
   - fix_database_schema.sql (pour ajouter la colonne account_name)
   - grant_permissions.sql (pour accorder les permissions à zalint)

4. Redémarrez le serveur Node.js
